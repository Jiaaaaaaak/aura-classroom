# 文案與溝通語調規範 (UX Writing & Tone of Voice)

文字是 UI 的另一種「介面」。在 SELf-corner 中，語調必須具備心理諮詢式的共情能力。

## 1. 語調核心 (Core Tone)

*   **共情 (Empathetic)**: 站在教師的角度，不責備，只建議。
*   **專業 (Authoritative)**: 引用 Satir 或 SEL 理論時，措辭精確、值得信賴。
*   **支持 (Supportive)**: 鼓勵嘗試，即使對話練習失敗也要給予正向反饋。

## 2. 寫作準則 (Writing Rules)

### 2.1 頁面標語 (Slogans)
*   **Do**: "準備好進入練習了嗎？" (溫暖邀請)
*   **Don't**: "開始測試" (具備壓力感)

### 2.2 錯誤訊息 (Error Messages)
*   **Do**: "哎呀，連線似乎有點不穩。請再試一次看看？" (共擔責任)
*   **Don't**: "ERROR 500: Server disconnected" (冰冷技術術語)

### 2.3 專家建議 (Expert Feedback)
*   **Structure**: 肯定亮點 -> 發現盲點 -> 行動導向建議。
*   **Phrasing**: "我注意到你使用了一致性的表達，這很棒！如果能加入更多關於「冰山下」的探討，或許效果會更顯著。"

## 3. 國際化 (i18n) 預留
*   預設使用 **繁體中文 (zh-TW)**。
*   專有名詞統一：
    - Satir -> 薩提爾。
    - Self-Awareness -> 自我覺察。
    - Scenario -> 練習情境。
